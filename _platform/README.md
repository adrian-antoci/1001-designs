# 1001 Designs

A Flutter project showcasing a collection of 1001 unique designs. This project serves as a library of beautiful, and practical designs that can be used as inspiration or as a starting point for your own Flutter applications.

## 🚀 Getting Started

This project is a starting point for a Flutter application.
To run this project locally, you will need to have Flutter installed on your machine. If you do not have Flutter installed, you can follow the instructions [here](https://flutter.dev/docs/get-started/install).

Once you have Flutter installed, you can clone this repository and run the following commands in your terminal:

```bash
flutter pub get
flutter run
```

## 📂 Project Structure

The project is organized into the following directories:

- `lib`: This is the main directory of the project, where all the Dart code is located.
  - `app`: This directory contains the main application logic, such as the main widget, routes, and exceptions.
  - `bloc`: This directory contains the business logic of the application, using the BLoC pattern.
  - `data`: This directory contains the data layer of the application, such as models, repositories, and data sources.
  - `widgets`: This directory contains the custom widgets used in the application.
- `assets`: This directory contains the static assets used in the application, such as images, fonts, and icons.
- `web`: This directory contains the web-specific files of the project.

## ✨ Features

- A collection of 1001 unique designs.
- A clean and organized codebase, following the BLoC pattern.
- A responsive design that works on both mobile and web.
